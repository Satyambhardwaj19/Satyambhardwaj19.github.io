TrainRadar — Live UK Train Tracker

A full-stack web application that tracks live UK trains on an interactive map. The core challenge: train APIs don't provide GPS coordinates — so I built an algorithm that estimates train position between stations using time-based interpolation. Users can search any UK station, view live departures, monitor journey progress, and set delay alerts.
